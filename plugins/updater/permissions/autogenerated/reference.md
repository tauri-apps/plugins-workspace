## Default Permission

This permission set configures which kind of
updater functions are exposed to the frontend.

#### Granted Permissions

The full workflow from checking for updates to installing them
is enabled.



- `allow-check`
- `allow-download`
- `allow-install`
- `allow-download-and-install`
